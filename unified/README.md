# @stem/unified

UnifiedLesson JSON **v0.1** — shared data contract for `diagram`, `game`, `movie`, and `webgpu`.

- **Types & validation**: `ts/schema.ts`, `ts/validate.ts`
- **Dry-run runtime** (no app wiring yet): `ts/runtime.ts`
- **Legacy detection / mapping notes**: `ts/migration/`

## Build

```bash
npm run build -w @stem/unified
```

Or from this directory: `npm run build`
